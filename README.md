
# Stock Price Analysis & Trend Prediction

## Project Overview
This project analyzes historical stock market data and predicts future stock trends using Machine Learning.

## Features
- Historical stock data analysis
- Data visualization
- Moving average analysis
- Trend prediction using Linear Regression
- Graph generation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yfinance

## Project Structure
```
Stock_Price_Analysis_Project/
│── data/
│── output/
│── stock_analysis.py
│── requirements.txt
│── README.md
```

## Installation
```bash
pip install -r requirements.txt
```

## Run the Project
```bash
python stock_analysis.py
```

## Dataset
The project uses Yahoo Finance stock data.

## Output
- Stock trend graphs
- Moving average chart
- Prediction results
